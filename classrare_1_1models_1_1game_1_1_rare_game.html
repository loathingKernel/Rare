◆ is_foreign()

bool is_foreign (   self)

Property to report if a game doesn't belong to the current account.

Checks if a game belongs to the currently logged in account. Games that require a network connection or remote authentication will fail to run from another account despite being installed. On the other hand, games that do not require network, can be executed, facilitating a rudimentary game sharing option on the same computer.

Returns
bool If the game belongs to another count or not
