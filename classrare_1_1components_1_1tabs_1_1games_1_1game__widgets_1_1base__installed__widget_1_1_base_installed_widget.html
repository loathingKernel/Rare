bool is_installed (   self)

Property to report if a game is installed.

This returns True if InstalledGame data have been loaded for the game or if the game is a game without assets, for example an Origin game.

Returns
bool If the game should be considered installed

◆ is_non_asset()

bool is_non_asset (   self)

Property to report if a Game doesn't have assets.

Typically, games have assets, however some games that require other launchers do not have them. Rare treats these games as installed offering to execute their launcher.

Returns
bool If the game doesn't have assets
